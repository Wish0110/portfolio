EduBridge

EduBridge

EduBridge, my final year project built with the MERN stack, aims to be a one-stop platform for streamlining the university application process. It mainly work with web crawling technology and Open AI API. It leverages advanced features like a Leaflet map to visualize universities, web crawling for gathering relevant information, and form autofilling to save you time. An AI bot provides personalized guidance, while the platform can even autogenerate recommendation letters and handle university application forms and payments. EduBridge also boasts a comprehensive forms management system and a dummy NSBM database for testing purposes, making it a valuable tool for students navigating the complexities of higher education applications.

DWANP Restaurants

DWANP Restaurants, our collaborative MERN stack project, is an online platform designed to revolutionize the dining experience. Node web crawler

This component of EduBridge is an automated web crawler built with Node.js. Its purpose is to systematically navigate and extract relevant information from university websites. This information could include course listx program details, application deadlines, tuition fees like all details about courses. The collected data can then be used to populate. This web crawler helps ensure EduBridge has up-to-date university data with scheduled crawling, enhancing the platform's value for students.
